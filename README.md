# Auto Dev Setup
Automatically sets up the following:
- Bash aliases
- Tmux config
- [KeVim Neovim config](https://github.com/kfalcetano/kevim)

Just clone this repo (generally to the home dir) and run `setmeup.sh`

> [!NOTE]
> This does not install any dependencies

## Required dependencies
- [exa](https://github.com/ogham/exa)
- [tmux](https://github.com/tmux/tmux)
- [fzf](https://github.com/junegunn/fzf)
- [Neovim](https://github.com/neovim/neovim)
  - `:checkhealth` for plugin dependencies
